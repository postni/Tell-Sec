Herzlich Willkommen bei Tell!


Heutzutage sind so gut wie alle Geräte in einem Unternehmen über ein Netzwerk miteinander verbunden.

Tell! ist ein neuartiges Tool zur Risikoanalyse. Egal ob Einsteiger oder Experte - für jeden Interessierten bietet die Software ein innovatives Layout mit entsprechend angepassten Inhalten. 

Somit übersiehst du keine Schnittstellen, die potenzielle Gefahren mit sich bringen. Diese Webseite begleitet dich durch die Software, um einen perfekten Start mit Tell! zu gewährleisten.

Probieren Sie es doch einfach aus!


Menüleiste


  Dieser Button ermöglicht Ihnen einen neuen Entwurfseditor zu öffnen.


  Hier können Sie Ihre erstellte(n) Systemlandschaft(en) speichern.


  Bereits erstellte Systemlandschaften können Sie hier öffnen oder hinzufügen.


  Wenn Sie mehrere Systemlandschaften in einem Entwurfseditor erstellt haben, können Sie hier die Seite blättern.


  Tell! bietet Ihnen die Suche nach bestimmten Inhalten in Ihrem Dokument.


  Hier gelangen Sie auf About, Hilfe und könnnen das Tool schließen.



Wichtigste Funktionen


Netzwerkscan

Mit dem Netzwerkscan erhalten Sie in einem Entwurfseditor eine grobe Übersicht über die gescannten Netzwerke und vorhandene Geräte im Unternehmen. Hierbei können Sie bei Bedarf die entstandene Netzwerkstruktur manuell anpassen und erweitern. Dabei wird jedoch vorausgesetzt, dass das Endgerät auf dem die Software läuft, mit dem gewünschten Netzwerk verbunden ist.

Manueller Netzwerkplan

Da Netzwerke sehr komplex sein können, und nicht zwangsläufig durch ein Endgerät erreichbar sein müssen, können Sie solche Netzwerksegmente auch zusätzlich komplett eigenständig modellieren. Diese manuelle Variante des Modellierens können Sie auch verwenden, wenn einzelne Komponenten bei dem Netzwerkscan nicht gefunden wurden. Sie können per Klick ganz einfach und unkompliziert Ihren (Produktions-) Netzwerk nachbilden. Hierdurch kann das Scannen des Netzwerks ergänzt oder vollständig ersetzt werden.

Hilfe bei der Identifizierung unbekannter Geräte

Falls der Netzwerkscan ein Netzwerkelement gefunden, aber nicht erkannt hat, symbolisiert ein Fragezeichen an entsprechender Stelle das Problem. Klicken Sie mit Ihrem Cursor auf das Objekt, so erscheint eine Infobox mit grundlegenden, bekannten Informationen, wie beispielsweise IP- und MAC-Adresse. Anhand dieser Informationen erhalten Sie, um welches Gerät im System es sich tatsächlich handelt. Anschließend können Sie per Tastatureingabe fehlende Information ergänzen. Ferner können auch weitere Netzwerkelemente – wie bspw. Benutzer, Maschinen, Netzwerke, Konnektoren oder technische Komponenten – manuell hinzugefügt werden, falls der Netzwerkscan im Betrieb nicht alles lückenlos erkennen könnte.

Risikoanalyse

Bei vollständig ausgearbeitetem Netzwerkplan können alle bekannten Risikoquellen identifiziert werden. An den Identifizierten „Problemstellen“ wird hier eine Risikobeschreibung inklusive möglicher Lösungsvorschläge angezeigt. Die Risikobewertung zeigt Ihnen die erste Auswertung der Systemlandschaft an, diese können Sie mit einem Schieberegler zusätzlich mit Wahrscheinlichkeit- und Schadenshöhe individuell anpassen.


Supportanfrage


Hier können Sie uns dabei unterstützen, Tell! weiter zu verbessern.

Bitte beachten Sie, dass wir auf solche Supportanfragen nicht persönlich antworten können. Dennoch ist jede Information unserer User für uns sehr aufschlussreich und hilfreich. Alle Suportanfragen werden von uns aufmerksam gelesen und jeweils an die richtigen Stellen weitergeleitet, um sie effektiv zu bearbeiten.

Sollten Sie eine direkte Frage, dann schreiben Sie und eine E-Mail an support@tell.de Wir sind bemüht, jede Frage zu technischen Themen zu beantworten. Falls Sie eine Frage zur Bedienung der Tell! haben, wenden Sie zunächst an die Suchfunktion hier im Handbuch. Sollten Sie keine Antwort finden, benachrichtigen Sie uns bitte über Ihr genaues Bedienproblem. Wir werden natürlich auch in unserem Handbuch weitere Nachbesserungen führen.
