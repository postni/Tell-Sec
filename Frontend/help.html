Herzlich Willkommen bei Tell!


Heutzutage sind so gut wie alle Geräte in einem Unternehmen über ein Netzwerk miteinander verbunden.

Tell! ist ein neuartiges Tool zur Risikoanalyse. Egal ob Einsteiger oder Experte - für jeden Interessierten bietet die Software ein innovatives Layout mit entsprechend angepassten Inhalten. 

Somit übersiehst du keine Schnittstellen, die potenzielle Gefahren mit sich bringen. Diese Webseite begleitet dich durch die Software, um einen perfekten Start mit Tell! zu gewährleisten.

Probieren sie es doch einfach aus!


Wichtigste Funktionen


Netzwerkscan

Mit dem Netzwerkscan erhalten Sie in einem Entwurfseditor eine grobe Übersicht über die gescannten Netzwerke und vorhandene Geräte im Unternehmen. Hierbei können Sie bei Bedarf die entstandene Netzwerkstruktur manuell anpassen und erweitern. Dabei wird jedoch vorausgesetzt, dass das Endgerät auf dem die Software läuft, mit dem gewünschten Netzwerk verbunden ist.

Manueller Netzwerkplan

Da Netzwerke sehr komplex sein können, und nicht zwangsläufig durch ein Endgerät erreichbar sein müssen, können Sie solche Netzwerksegmente auch zusätzlich komplett eigenständig modellieren. Diese manuelle Variante des Modellierens können Sie auch verwenden, wenn einzelne Komponenten bei dem Netzwerkscan nicht gefunden wurden. Sie können mit Drag & Drop ganz einfach und unkompliziert Ihren (Produktions-) Netzwerk nachbilden. Hierdurch kann das Scannen des Netzwerks ergänzt oder vollständig ersetzt werden.

Hilfe bei der Identifizierung unbekannter Geräte

Falls der Netzwerkscan ein Netzwerkelement gefunden, aber nicht erkannt hat, symbolisiert ein Fragezeichen an entsprechender Stelle das Problem. „Hovern“ Sie mit Ihrem Cursor über diesem Objekt, so erscheint eine Infobox mit grundlegenden, bekannten Informationen, wie beispielsweise IP- und MAC-Adresse. Anhand dieser Informationen erhalten Sie, um welches Gerät im System es sich tatsächlich handelt. Anschließend können Sie per Tastatureingabe fehlende Information ergänzen. Ferner können auch weitere Netzwerkelemente – wie bspw. Benutzer, Maschinen, Netzwerke, Konnektoren oder technische Komponenten – manuell hinzugefügt werden, falls der Netzwerkscan im Betrieb nicht alles lückenlos erkennen könnte.

Risikoanalyse

Bei vollständig ausgearbeitetem Netzwerkplan können alle bekannten Risikoquellen identifiziert werden. An den Identifizierten „Problemstellen“ wird hier eine Risikobeschreibung inklusive möglicher Lösungsvorschläge angezeigt. Nach dem Netzwerkscan können Sie durch einen Klick auf den Analyse-Button anhand von roten Ausrufezeichen potenzielle Gefahrenquelle aufgezeigt werden. Durch das „hovern“ mit dem Cursor so über das Ausrufezeichen erscheint erneut ein Tooltip mit Informationen über denkbaren Risiken. Im Experten-Modus werden die Informationen detaillierter – für Laien nicht mehr verständlich – dargestellt.
